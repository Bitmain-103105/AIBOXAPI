# 1.5.1.2 删除用户（delete\_users）

**功能描述：**

删除一个或多个用户。

如果删除列表中的某个用户ID不存在，则会跳过删除此用户，继续执行其它用户的删除。

**参数描述（JSON格式）：**

\*\*\*\*

| **一级参数** | 二级参数 | 类型 | 选择 | 说明 |
| :--- | :--- | :--- | :--- | :--- |


| method |  | string | **必选** | delete\_users |
| :--- | :--- | :--- | :--- | :--- |


| messageId |  | string | 可选 | MQ消息的唯一标识ID |
| :--- | :--- | :--- | :--- | :--- |


| userid\_list |  | list | **必选** | 用户id列表 |
| :--- | :--- | :--- | :--- | :--- |


| extend\_params |  | dict | 可选 | 扩展字段 |
| :--- | :--- | :--- | :--- | :--- |
